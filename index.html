my-first-gitbook

项目初始化

//创建如下目录结构
my-gitbook/
├── README.md  //书籍介绍
└── SUMMARY.md //书籍目录结构

gitbook init //创建 SUMMARY.md 中的目录结构
gitbook serve //编译 && 预览

gitbook build // 编译

部署说明

  • master: 保存书籍的源码
  • gh-pages: 发布后的静态页面，便于页面访问
